# 项目目标与里程碑（v2 修订）

修订日期：2026-06-12 ｜ 交付期限：≈2026-06-25（面试）
取代原始目标（"崎岖路面车身自稳演示"，已于 v3 超额达成，见 01 §5）。

## 1. 项目定位（一句话）

**双仿真器验证的六轮主动悬挂底盘：roll/pitch/z 三自由度车身控制
+ 轮地接触保持 + 控制延迟归因实验（Gazebo → Isaac Lab）+ RL 对照。**

从"调平演示"升级为"全功能车身控制 + 工程方法论展示"：
- 功能层：调平（已完成）→ 高度调节 → 接触保持，覆盖工业主动悬挂三大经典功能
- 方法层：每项功能有量化验收、失效实验定界、安全包络（已有传统延续）
- 视野层：同一 core 模型跨仿真器复现，kHz 控制环验证延迟归因，RL 对照，
  2-DOF 下一代构型设计研究（吸收 TeCVP，见 §4）

## 3. 非目标（明确砍掉的范围）

- **2-DOF 腿全量实现**：只做 M3 设计研究。论文级工作量（全身运动学+阻抗+
  腿轮切换状态机）与交付期冲突，且会挤掉 Isaac Lab 主线。
- **阻抗控制实现**：v3 平衡点平移即等效刚度控制（阻抗的位置项），物理弹簧
  承担快动力学——叙事层吸收，不另写代码（v1 力控失败教训仍有效）。
- **腿式步态 / 腿轮模式切换**：构型不支持，无意义。
- **六轮载荷均衡**：列为 M7 之后的可选项，时间不够即放弃。
